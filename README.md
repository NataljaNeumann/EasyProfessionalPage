# Easy Professional Page
This project helps you to easily make professionally looking web pages, by providing a CSS that you can include in your web pages that will style the elements in your pages. The CSS comes with a calm color scheme. You need to define colors of your page yourself. Of course, you can ask an AI for providing you a color scheme that is adapted to your needs.

Especially the CSS supports three different color schemes:
- For screens with light color preference
- For screens with dark color preference
- For printing

You need to define all three color schemes, as well as the sizes of margins and elements, then the CSS will get its power for styling the element on your page.

The sizes and colors are stored in variables, so you don't need to specify them on each element.

Additionally the CSS supports color scheme for people that need specific colors, if this has been set in browser.

एषा परियोजना व्यावसायिकरूपेण दृश्यमानानि जालपुटानि सहजतया निर्मातुं साहाय्यं करोति, एकं CSS प्रदातुं यत् भवान् स्वस्य जालपृष्ठेषु समावेशयितुं शक्नोति यत् भवतां पृष्ठेषु तत्त्वानां शैलीं करिष्यति CSS शान्तवर्णयोजनया सह आगच्छति । भवद्भिः स्वपृष्ठस्य वर्णाः स्वयमेव परिभाषितव्याः । अवश्यं, भवान् एआइ इत्यस्मै पृच्छितुं शक्नोति यत् सः भवतः आवश्यकतानुसारं वर्णयोजनां प्रदातुं शक्नोति ।

विशेषतः CSS त्रीणि भिन्नानि वर्णयोजनानि समर्थयति:
- हल्के वर्णप्राथमिकतायुक्तानां पटलानां कृते
- गहरे वर्णप्राथमिकतायुक्तानां पटलानां कृते
- मुद्रणार्थम्

भवद्भिः त्रयः अपि वर्णयोजनाः परिभाषितव्याः, तथैव मार्जिनस्य, तत्त्वानां च आकाराः, ततः CSS भवतः पृष्ठे तत्त्वस्य स्टाइल् कर्तुं स्वस्य शक्तिं प्राप्स्यति ।

आकाराः वर्णाः च चरयोः संगृहीताः सन्ति, अतः भवद्भिः प्रत्येकस्मिन् तत्त्वे तान् निर्दिष्टुं आवश्यकता नास्ति ।

अतिरिक्तरूपेण CSS तेषां जनानां कृते वर्णयोजनां समर्थयति येषां विशिष्टवर्णानां आवश्यकता वर्तते, यदि एतत् ब्राउजरे सेट् कृतम् अस्ति ।

सुलभतया ज्ञातुं शक्यते इति वर्णनं समाविष्टम् अस्ति ।


# Shafin ƙwararru mai sauƙi
Wannan aikin yana taimaka muku wajen ƙirƙirar shafukan yanar gizo masu ƙwarewa cikin sauƙi, ta hanyar samar da CSS da za ku iya haɗawa a cikin shafukan yanar gizonku wanda zai tsara abubuwan da ke cikin shafukanku. CSS ya zo da tsarin launi mai nutsuwa. Kuna buƙatar ayyana launuka na shafin ku da kanku. Tabbas, zaku iya tambayar AI don samar muku da tsarin launi wanda ya dace da bukatun ku.

Musamman CSS yana goyan bayan tsarin launi daban-daban guda uku:
- Don fuska tare da zaɓin launi mai haske
- Don allo tare da zaɓin launi mai duhu
- Domin bugu

Kuna buƙatar ayyana duk tsarin launi guda uku, da kuma girman girman gefe da abubuwa, sannan CSS za ta sami ƙarfin sa don yin salo a shafinku.

Ana adana masu girma da launuka a cikin masu canji, don haka ba kwa buƙatar saka su akan kowane kashi.

Bugu da ƙari, CSS na goyan bayan tsarin launi ga mutanen da ke buƙatar takamaiman launuka, idan an saita wannan a cikin mashin din.

Ya ƙunshi bayanin mai sauƙin koya.

# Ukurasa Rahisi wa Kitaalamu
Mradi huu hukusaidia kutengeneza kurasa za wavuti zinazoonekana kitaalamu kwa urahisi, kwa kutoa CSS ambayo unaweza kujumuisha katika kurasa zako za wavuti ambazo zitatengeneza vipengele kwenye kurasa zako. CSS inakuja na mpango wa rangi tulivu. Unahitaji kufafanua rangi za ukurasa wako mwenyewe. Kwa kweli, unaweza kuuliza AI kwa kukupa mpango wa rangi ambao umebadilishwa kulingana na mahitaji yako.

Hasa CSS inasaidia miradi mitatu tofauti ya rangi:
- Kwa skrini zilizo na upendeleo wa rangi nyepesi
- Kwa skrini zilizo na upendeleo wa rangi nyeusi
- Kwa uchapishaji

Unahitaji kufafanua mipango yote mitatu ya rangi, pamoja na ukubwa wa pembezoni na vipengele, kisha CSS itapata nguvu zake kwa kuweka kipengele kwenye ukurasa wako.

Saizi na rangi huhifadhiwa katika anuwai, kwa hivyo hauitaji kuzibainisha kwenye kila kipengele.

Zaidi ya hayo CSS inasaidia mpango wa rangi kwa watu wanaohitaji rangi maalum, ikiwa hii imewekwa katika kivinjari.

Ina maelezo ambayo ni rahisi kujifunza.
# صفحة احترافية سهلة
يساعدك هذا المشروع على إنشاء صفحات ويب احترافية بسهولة، من خلال توفير لغة CSS تُدمج في صفحات الويب، وتُنسق عناصرها. تتميز لغة CSS بمخطط ألوان هادئ. ما عليك سوى تحديد ألوان صفحتك بنفسك. بالطبع، يمكنك الاستعانة بالذكاء الاصطناعي لتزويدك بمخطط ألوان مُخصص لاحتياجاتك.

تدعم لغة CSS ثلاثة مخططات ألوان مختلفة:
- للشاشات ذات الألوان الفاتحة
- للشاشات ذات الألوان الداكنة
- للطباعة

ما عليك سوى تحديد مخططات الألوان الثلاثة، بالإضافة إلى أحجام الهوامش والعناصر، لتتمكن لغة CSS من تنسيق عناصر صفحتك.

يتم تخزين الأحجام والألوان في متغيرات، لذا لا داعي لتحديدها لكل عنصر.

بالإضافة إلى ذلك، تدعم لغة CSS مخططات الألوان لمن يحتاجون إلى ألوان مُحددة، إذا تم ضبط ذلك في المتصفح.

يحتوي على وصف سهل الفهم.


# صفحه حرفه‌ای آسان
این پروژه به شما کمک می‌کند تا به راحتی صفحات وب حرفه‌ای بسازید، با ارائه CSS که می‌توانید در صفحات وب خود بگنجانید و عناصر صفحات شما را استایل‌دهی کند. CSS با یک طرح رنگی ملایم ارائه می‌شود. شما باید خودتان رنگ‌های صفحه خود را تعریف کنید. البته، می‌توانید از یک هوش مصنوعی بخواهید که طرح رنگی متناسب با نیازهای شما را در اختیارتان قرار دهد.

به خصوص CSS از سه طرح رنگی مختلف پشتیبانی می‌کند:
- برای صفحه نمایش‌هایی با ترجیح رنگ روشن
- برای صفحه نمایش‌هایی با ترجیح رنگ تیره
- برای چاپ

شما باید هر سه طرح رنگی و همچنین اندازه حاشیه‌ها و عناصر را تعریف کنید، سپس CSS قدرت خود را برای استایل‌دهی به عنصر در صفحه شما به دست می‌آورد.

اندازه‌ها و رنگ‌ها در متغیرها ذخیره می‌شوند، بنابراین نیازی به تعیین آنها برای هر عنصر ندارید.

علاوه بر این، CSS از طرح رنگی برای افرادی که به رنگ‌های خاص نیاز دارند، در صورتی که در مرورگر تنظیم شده باشد، پشتیبانی می‌کند.

شامل توضیحاتی آسان برای یادگیری است.
